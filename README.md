# 📌 Project Overview

The **BlinkIT Grocery Sales Dashboard** is a Power BI project designed to analyze grocery sales data and extract meaningful business insights.

This dashboard helps businesses understand:

✔ Sales performance
✔ Product distribution
✔ Outlet performance
✔ Customer ratings

Using **interactive visualizations**, the project enables better **data-driven decision making**.

---

# 🎯 Project Objectives

🔹 Analyze overall sales performance
🔹 Identify top performing outlets
🔹 Understand product category trends
🔹 Evaluate outlet size and location impact
🔹 Provide actionable business insights

# 📊 Dashboard Visualizations

* Small outlets
* Medium outlets
* Large outlets

### 🌍 Sales by Location Tier

Analyzes performance across:

* Tier 1 Cities
* Tier 2 Cities
* Tier 3 Cities

### 🍔 Product Category Rating

Displays **average ratings for each product type**.

### 🧈 Fat Content Distribution

Shows distribution of:

* Low Fat products

### 🏪 Outlet Type Performance

* Grocery Stores

* Item Table

* Item Fat Content
* Outlet Year

This structure improves **query performance and data analysis efficiency**.
